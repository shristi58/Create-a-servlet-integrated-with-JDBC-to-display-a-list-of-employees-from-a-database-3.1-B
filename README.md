# EmployeeServletApp

A simple Java Servlet + JDBC application to display and search employees.

## Features
- View all employees
- Search employee by ID
- Connected to MySQL database

## Setup
1. Import into Eclipse or IntelliJ as a Dynamic Web Project.
2. Update database credentials in `DBConnection.java`.
3. Deploy on Apache Tomcat (v9+).
4. Access at: `http://localhost:8080/EmployeeServletApp/`
